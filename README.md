## 📝 Sobre o Projeto
Este script em Python foi desenvolvido para facilitar a visualização do status de um reservatório de água. Utilizando a biblioteca `colorama`, o sistema fornece alertas coloridos no terminal, permitindo uma rápida identificação da situação atual, indo do nível crítico ao alerta de nível máximo.

## 🚀 Funcionalidades
- [x] **Monitoramento em tempo real**: Exibe status baseados em níveis (1 a 5).
- [x] **Feedback Visual**: Uso de cores intuitivas para cada situação.
- [x] **Código Limpo**: Estruturado com funções para fácil manutenção.

## 🛠 Tecnologias Utilizadas
- **Linguagem**: [Python](https://www.python.org/)
